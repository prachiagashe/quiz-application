# quiz-application
A Java Quiz Application that allows users to answer multiple-choice questions, calculates scores, percentages, and grades using Core Java and OOP concepts.

# Java Quiz Application

## Features
- Multiple Choice Questions
- Score Calculation
- Percentage Calculation
- Grade Generation
- Pass/Fail Status

## Technologies
- Java
- OOP
- Arrays
- Loops
- Conditional Statements

## Future Enhancements
- Java Swing GUI
- MySQL Database
- Timer-Based Quiz
